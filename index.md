# Política de Privacidad – KillScroll

Última actualización: 22/01/2026

KillScroll es una aplicación de productividad que funciona completamente de forma local en el dispositivo del usuario.

## Permisos utilizados

### Cámara
La cámara se utiliza exclusivamente para detectar movimientos físicos (por ejemplo, flexiones) como condición para desbloquear aplicaciones.
Las imágenes NO se almacenan, NO se graban y NO se envían a ningún servidor.

### Accesibilidad
El permiso de accesibilidad se utiliza únicamente para detectar qué aplicación está activa y aplicar reglas de bloqueo definidas por el usuario.
KillScroll NO lee contenido de otras aplicaciones.

### Estadísticas de uso
Se utilizan estadísticas de uso para identificar la aplicación actualmente en uso.
Los datos se procesan únicamente en el dispositivo.

## Datos recopilados
KillScroll NO recopila, almacena ni comparte datos personales.

## Compartición de datos
KillScroll NO comparte datos con terceros.
